# Find min/max number in array

## Find max
Given an array of integers find the biggest integer.

---
Sample input:
``` java
[1,2,1,51,2]
```
Sample output:
``` java
51
```
---
Sample input:
``` java
[1000,2,200, -5124]
```
Sample output:
``` java
1000
```
---

## Find min
Given an array of integers find the smallest integer.

---
Sample input:
``` java
[1,2,1,51,2]
```
Sample output:
``` java
1
```
---
Sample input:
``` java
[1000,2,200, -5124]
```
Sample output:
``` java
-5124
```
